# Orel Abecassis - Premium Landing Page

A modern, dark-themed landing page built with Vue 3, Tailwind CSS, and Motion.

## Features

-   **Premium Dark Design**: Elegant UI with glassmorphism and gold accents.
-   **Responsive**: Fully optimized for mobile, tablet, and desktop.
-   **Animations**: Smooth entrance animations and micro-interactions using `@vueuse/motion`.
-   **Components**:
    -   Hero Section with gradient text and background effects.
    -   Services Section with hover effects.
    -   Portfolio Grid with image reveals.
    -   Contact Form with validation simulation.
    -   Footer with social links.

## Tech Stack

-   [Vue 3](https://vuejs.org/)
-   [Tailwind CSS](https://tailwindcss.com/)
-   [Vite](https://vitejs.dev/)
-   [Lucide Vue](https://lucide.dev/guide/packages/lucide-vue-next) (Icons)
-   [VueUse](https://vueuse.org/) (Utilities)

## Setup

1.  Install dependencies:
    ```bash
    npm install
    ```

2.  Run development server:
    ```bash
    npm run dev
    ```

3.  Build for production:
    ```bash
    npm run build
    ```
